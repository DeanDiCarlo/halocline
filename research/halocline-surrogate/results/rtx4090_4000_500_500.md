# RTX 4090 U-FNO Surrogate Benchmark

This benchmark trains a U-FNO surrogate against the synthetic Python physics oracle, then evaluates against held-out oracle scenarios.

## Run Setup

- Hardware: RTX 4090 RunPod instance
- Training scenarios: 4,000 synthetic oracle scenarios
- Validation scenarios: 500
- Test scenarios: 500
- Epochs: 30
- Training config: `configs/ufno_h100_fast.yaml`
- Dataset source: generated from `configs/sampling.yaml`

## Runtime Benchmark

| Surrogate batch=64 | 0.2337 | 48.42x |

| Surrogate batch=2048 | 0.0648 | 174.57x |

## Interpretation

The run demonstrates a meaningful batched inference speedup over the Python physics oracle while improving substantially over the small CPU-scale smoke run. The validation loss decreased monotonically through 30 epochs, so this run did not show obvious overfitting within the logged training window.

This result should be framed carefully: accuracy is measured against the synthetic Python physics oracle, not field observations. The oracle is a simplified 2D steady sharp-interface model, not a calibrated Biscayne aquifer model.
